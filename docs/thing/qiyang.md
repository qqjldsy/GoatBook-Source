# 岐阳市
## 基本信息
* 中文名：岐阳市  
* 外文名：Qiyang Province  
* 所属地区：茕羊民国  
* 创立时间：2026年4月8日左右(可能先期也有建立，但具体不确定)  
* 主要职责：缓解群员的政压抑  
* 现任省长：IM星闪  
## 历史沿革
本质上是对前几个见证特区，尤其是对**晟日省**开明的见证政策的沿袭。  
前期称作星闪市，后期更名为岐阳市。  
## 特色政策
本群允许开放见证，一切见证行为在这里不受约束。
