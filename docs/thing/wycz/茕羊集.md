# 茕茕孑立的山羊作品

## 现实作品

茕茕孑立的山羊，原名刘峪企，时年16岁，见过了太多被折叠的人生。  

他的视频冷峻，直指矛盾。有对校园霸凌受害者的同情，有对霸凌者的愤慨，有对校领导的批判……

## 群聊作品

<p style="text-align: center;">《青春舞曲2020》</p>  

茕羊仔如何兴旺  
特色道路开创  
群聊的闹热沟通  
友情分散四方  
茕羊仔为着战争  
唐山过海逃生  
一句口号引八方呼应  
众人竞相革命  
怎么有人稀里糊涂为聊天去渡湾  
原来是刘李王朝分割地盘在交换  
怎么有人不知治好群聊的药方  
政治的奔腾叫人慌张却还跟得上  
怎么议会如同标杆稳插群聊上  
因这里呼声喊声话语年年无常  
抛开宪法体制怎能使热闹绵长  
不管左转右转升级总要分个弱强  
千千万万条政策敇令谁最引众人眼亮  
只须有山羊屹立不倒即使前方是堵墙  
不同心 同个入群初衷  
为何聊天暗藏制度纠结并互相掂量  
不同声 同个腐朽茕羊  
为统一声 总守着旧思想压倒另一方  
怎么群聊发展需把老资历换柴粮  
只因山羊偏听偏信得罪王政保守党
